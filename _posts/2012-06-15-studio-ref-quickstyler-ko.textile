QuickStyler view 는 스타일 관련 속성들을 쉽게 수정할 수 있는 UI 를 제공합니다. QuickStyler 는 여러개의 section 으로 구성되어 있으며 컴포넌트에 따라 나타나는 section 이 다를 수 있습니다.

!http://s17.postimage.org/btry0z2lb/quickstyler.png!
h1. Text section
 
!http://s13.postimage.org/ya6j7odbb/text_section.png!
 
*1: Font Family* 
Font를 선택합니다. 마지막 ADD... 를 선택하면 *Preferences 의 Font Families page* 가 뜨고 선택할 수 있는 Font 의 종류를 편집할 수 있습니다.
 
*2: Font Size*
Font 크기를 pixel 단위로 선택합니다.
 
*3: Align Left*
텍스트를 왼쪽으로 정렬합니다.
 
*4: Align Center*
텍스트를 중앙으로 정렬합니다.
 
*5: Align Right*
텍스트를 오른쪽으로 정렬합니다.
 
*6: Bold*
텍스트에 bold 스타일을 적용합니다.
 
*7: Italic*
텍스트에 italic 스타일을 적용합니다.
 
*8: Underline*
텍스트에 underline 스타일을 적용합니다.
 
*9: Line Height*
텍스트의 줄간격을 변경합니다. 숫자 또는 pixel 값을 적용할 수 있습니다.
 
h1. Color Section
 
Color Section 에서 배경색, 텍스트색, border 색상을 변경할 수 있습니다. 변경할 색상을 선택한 후 하단의 Swatchs, Solid, Graident, Image 중 한곳에서 색상을 변경합니다.
 
!http://s15.postimage.org/amspgttxn/color_section.png!
 
*1: Background Color" 
배경색을 선택합니다. Reset 버튼으로 배경색을 초기화할 수 있습니다.
 
*2: Foreground Color"
텍스트색상을 선택합니다. Reset 버튼으로 텍스트색상을 초기화할 수 있습니다.
 
*3: Border Color"
Border 색을 선택합니다. Reset 버튼으로 border 색을 초기화할 수 있습니다.
 
h3. Swatches
 
미리 정의된 swatch 에서 색상을 선택합니다. 또한 색상을 ==투명색(transparent)== 으로 변경할 수 있습니다.
 
*4*
Swatch 의 종류를 선택합니다.
 
*5*
Swatch 를 추가합니다. "이곳":http://www.colorzilla.com/firefox/palettes.html 에서 swatch 정의 파일을 받을 수 있습니다.
 
*6*
선택된 색상을 transparent 로 변경합니다.
 
h4. Solid
 
단일 색상을 선택할 수 있습니다.
 
!http://s17.postimage.org/bvzvo2xun/solid_color.png!
 
*1*
이곳에서 원하는 색상을 선택합니다.
 
*2*
1번에서 출력되는 색상의 기본 색상을 지정합니다. 
 
*3,4,5*
선택된 색상값을 보여주고 직접 변경할 수 있습니다.
 
*6*
선택된 색상값은 web color 형식으로 보여주고 변경할 수 있습니다.
 
*7*
현재 선택된 색상을 나타냅니다.
 
*8*
색상값을 모니터에서 직접 선택합니다.
 
h4. Gradient
 
Gradient 색상을 선택할 수 있습니다.
 
!http://s16.postimage.org/t6nkveg2t/gradient_color.png!
 
*1*
현재 선택된 stop 의 색상을 선택합니다.
 
*2*
1 의 기본 색상을 변경합니다.
 
*3,4,5*
현재 선택된 stop 의 색상 값을 보여주고 변경할 수 있습니다.
 
*6: STOP*
현재 선택된 stop 의 위치를 % 단위로 보여주고 변경할 수 있습니다.
 
*7: WebColor*
현재 선택된 stop 의 색상을 web color 형식으로 보여주고 변경할 수 있습니다.
 
*8: Type*
Gradient type 을 linear 와 radial 중 선택할 수 있습니다.
 
*9: Direction*
Gradient type 이 linear 인 경우 나타납니다. Gradient 방향을 Vertical, Horizontal, UpwardDiagonal, DownwardDiagonal 중 하나로 선택할 수 있습니다.
 
*10*
현재 선택된 stop 의 색상을 보여줍니다.
 
*11*
현재 적용된 gradient 를 미리보기로 보여줍니다. 이영역에 클릭하면 stop 이 추가됩니다.
 
*12*
선택된 stop 을 나타냅니다. stop 을 위쪽이나 아래쪽으로 마우스 드래그하면 stop 이 제거됩니다.
 
*13*
선택되지 않은 stop 을 나타냅니다. stop 을 위쪽이나 아래쪽으로 마우스 드래그하면 stop 이 제거됩니다.
 
*14*
graident stop 의 위치를 반전시킵니다.
 
*15*
stop 색상값을 모니터에서 직접 선택합니다.
 
h4. Image
 
배경 이미지를 지정합니다.
 
!http://s17.postimage.org/a374hj1sv/image.png!
 
*1: Background Image*
배경 이미지 파일의 경로를 지정합니다.
 
*2: Repeat*
배경 이미지가 반복되는 방식을 정의합니다.
