Class tau.ui.SceneController 

Object
   |
   +--tau.rt.EventDelegator
      |
      +--tau.ui.Controller
         |
         +--tau.ui.SceneController
Direct Known Subclasses:
tau.ui.DashboardController, tau.ui.TableSceneController

Navigator에서 메인 scene과 다음 두가지 추가적인 하부 scene을 관리하는데 사용되는 기본 컨트롤러이다. 

 NavigationBar - 스크린 네비게이션과 추가적인 Super-context 메뉴를 컨트롤하는 scene
 Content - 컨트롤러에 관리되는 메인 컨텐트 scene
 ToolBar - 메인 컨텐트의 밖에서 Super-context 메뉴를 컨트롤하는 데 필요한 scene

Constructor Summary
new tau.ui.SceneController (opts) 생성자, 주어진 Scene객체 또는 Scene파일을 이용하여 객체를 생성한다.
Method Summary
propagateEvent(e) 이벤트 Subsystem메소드를 Override한 메소드이다.
doStart() Controller Lifecycle 메소드를 Override한다.
doDraw() SceneController가 소유하고 있는 Scene의 Drawing작업을 수행한다.
getNavigationBar() 현재 컨트롤러에 설정된 Navigation Bar를 반환한다.
setNavigationBar(navBar) 현재 컨트롤러에 Navigation Bar에 설정한다.
getToolBar() 현재 컨트롤러에 설정된 Tool Bar를 반환한다.
setToolBar(toolBar) 현재 컨트롤러에 Tool Bar를 설정한다.
getScene() 컨트롤러에 연관된 scene을 반환한다.
setScene(scene) 컨트롤러에 scene을 설정한다.
clearScene() 컨트롤러에 scene을 제거한다.
destroy() scene과 DOM을 클리어 하기 위한 컨트롤러의 destroy 콜백을 처리한다.
getDOM() tau.ui.SceneController 인스턴스의 루트 DOM element는 Scene의 DOM element이다.
loadScene() 컨트롤러의 로드 scene 액션 콜백 함수 관리되는 scene의 컨텐츠를 수정하기 위해서는 오버라이드해야한다.
sceneLoaded() 컨트롤러의 sceneLoaded 콜백 함수.
sceneDrawn() 컨트롤러의 sceneDrawn 콜백 함수.

                    tau.ui.SceneController.propagateEvent(e)
                    
                  

                  

                    이벤트 Subsystem메소드를 Override한 메소드이다.
SceneController에서 SCENELOADED이벤트가 발생하면 SceneController에 있는 시스템
메소드인 doDraw를 호출한다.

Scene은 Asynchronous하게 로딩되기 때문에 Scene의 로딩이 완료되기 전에 Draw가 진행
되면 화면이 정상적으로 그려지지 않는다. 따라서 tau.rt.Event.SCENELOADED
이벤트를 받은 다음 이후 과정인 Draw를 수행해야 한다.

                    tau.ui.SceneController.doDraw()
                    
                  

                  

                    SceneController가 소유하고 있는 Scene의 Drawing작업을 수행한다. 먼저 Drawing

                    tau.ui.SceneController.presentModal(ctrl, opts)
                    
                  

                  

                    Activate modal scene and displays it on top of dimmed layer. You can
specify various options along with the SceneController to be
used as ModalController.

The options which is plain JSON object can be used are as follow: 


{
  layout: 'FULL'|'PAGE'|'FORM'
         - specifies modal layout. default is 'FULL'

  animate: 'vertical' | 'horizontal'
         - indicates that animation will be applied and it's value is
           animation type to be used.
}
